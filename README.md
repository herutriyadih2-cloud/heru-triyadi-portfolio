# Professional Portfolio Website

A modern, responsive, and visually appealing personal portfolio website built with **Vue 3**, **Vite**, and **Tailwind CSS**.

## 🚀 Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **Smooth Animations:** Powered by **AOS (Animate on Scroll)** for a polished user experience.
- **Modern UI:** Styled with **Tailwind CSS** following Material Design principles.
- **Dynamic Components:** Modular architecture using Vue 3 Composition API.
- **Automated Deployment:** Ready for GitHub Pages deployment via GitHub Actions.

## 🛠️ Tech Stack

- **Frontend Framework:** [Vue.js 3](https://vuejs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide Vue Next](https://lucide.dev/)
- **Animations:** [AOS (Animate on Scroll)](https://michalsnik.github.io/aos/)

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   cd portfolio-profesional
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🚀 Deployment

This project is configured to deploy automatically to **GitHub Pages** whenever you push to the `main` branch.

1. Ensure your `vite.config.js` has the correct `base` path for your repository.
2. Push your changes to the `main` branch.
3. Check the **Actions** tab in your GitHub repository to monitor the deployment process.
